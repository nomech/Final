## Setup

1. Clone the repository
```bash
    git clone https://github.com/nomech/Final.git
```
2. Navigate to the project directory
```bash
cd Final
```
3. Open the Project Locally in live server 
 ```bash
 npm install -g live-server
 live-server

 ```
